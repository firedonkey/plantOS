What is PlantLab

A small plant lab that starts at home.

PlantLab turns a Raspberry Pi into a practical plant monitoring station. It keeps photos, sensor readings, and pump or light controls in one place, so each setup can grow from a simple experiment into a dependable care system.

Core Capabilities

Everything needed for the first real grow loop.

Camera

Visual Monitoring

Capture plant images over time and review growth changes without opening a terminal.

Sensors

Sensor Tracking

Follow moisture, temperature, humidity, pump state, and light state from one live view.

Cloud

Cloud Dashboard

Send Raspberry Pi readings to Cloud Run, then check status and controls from any browser.

How it works

Three steps from plant to dashboard.

01

Connect the Pi

Attach the camera, sensors, relay, light, and pump to the local device code.

02

Send readings

The Pi posts sensor data and images to the secure PlantLab backend.

03

Act with confidence

Use the dashboard to review trends, check images, and control light or watering.

Why PlantLab

Built for hands-on growing, not black-box automation.

Readable Python on the device.

Cloud history without losing local debugging.

Controls that wait for hardware acknowledgement.

A path from mock data to real plants.

Future Vision

A care system that learns each plant's rhythm.

Future PlantLab releases can add growth timelines, anomaly alerts, multi-plant rooms, smarter watering schedules, and AI-assisted plant notes.
